# Drupal 10 Event Registration Module

Custom event registration system for Drupal 10/11.

## Features (Planned)
- Event content type management
- User registration for events
- Capacity management
- CSV exports
- Email notifications

## Installation

1. Download the module into `web/modules/custom/event_registration`.
2. Enable the module via Drush:
   ```bash
   drush en event_registration -y
   ```
3. Configure permissions at `/admin/people/permissions`.
